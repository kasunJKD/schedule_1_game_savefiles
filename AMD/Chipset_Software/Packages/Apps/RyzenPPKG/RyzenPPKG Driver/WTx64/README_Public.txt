AMD Processor Power Management Support / AMD Ryzen Power plan
=============================================================
Version 8.0.0.13

Description:
------------

This driver provides processor power management Windows packaging settings for AMD Ryzen based Mobile and Desktop chipsets.
	
Operating System Supported:
--------------------------

* Microsoft Windows 10 64 bit
* Microsoft Windows 11 64 bit


Installation Instructions:
--------------------------

**NOTE**:  As with all driver installations on Windows, you need to login as Administrator or have administrator rights for your domain login.

To install the driver, copy the installation files to a temporary directory, then execute "AMDPPMSetting.exe -install", and wait for all command windows to close. 

 
Update Instructions:
--------------------

**NOTE**:  As with all driver installations on Windows, you need to login as Administrator or have administrator rights for your domain login.

To update the driver:

1. Copy the installation files to a temporary directory, then execute "AMDPPMSetting.exe -uninstall", and wait for all command windows to close.  
2. Copy the installation files to a temporary directory, then execute "AMDPPMSetting.exe -install", and wait for all command windows to close.  


Uninstall Instructions:
-----------------------

**NOTE**:  As with all driver installations on Windows, you need to login as Administrator or have administrator rights for your domain login.

To install the driver, copy the installation files to a temporary directory, then execute "AMDPPMSetting.exe -uninstall", and wait for all command windows to close.
